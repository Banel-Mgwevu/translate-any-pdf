STEP 3: INSTALL DEPENDENCIES
═══════════════════════════════════════════════════════════════════

Open Command Prompt or PowerShell in your project folder and run:

pip install googletrans==4.0.0rc1 defusedxml

Or if that doesn't work:
   python -m pip install googletrans==4.0.0rc1 defusedxml

Or with pip3:
   pip3 install googletrans==4.0.0rc1 defusedxml

═══════════════════════════════════════════════════════════════════
STEP 5: RUN THE TRANSLATOR
═══════════════════════════════════════════════════════════════════

METHOD 1 - GUI (Easiest):
──────────────────────────
python document_translator_gui.py

Then:
1. Click "Browse..." to select your input document
2. Choose output filename
3. Select target language
4. Click "Translate Document"


METHOD 2 - Command Line:
──────────────────────────
Basic syntax:
   python document_translator.py input.docx output.docx TARGET_LANG

Examples:

# Translate to Spanish
python document_translator.py mydoc.docx mydoc_spanish.docx es

# Translate to French  
python document_translator.py mydoc.docx mydoc_french.docx fr

# Translate to German
python document_translator.py mydoc.docx mydoc_german.docx de

# Translate from English to Japanese
python document_translator.py mydoc.docx mydoc_japanese.docx ja en


METHOD 3 - Create Sample First:
────────────────────────────────
# If you have Node.js installed:
node create_sample_document.js

# Then translate the sample:
python document_translator.py sample_document.docx translated.docx es
